Bien avant notre rentrée, nous avions à disposition une série de 18 quêtes : le Tech Challenge. Ces différentes quêtes avaient pour but d’apprendre aux wilders les bases du code pour ainsi atteindre un but final : créer une page web. ( Selon la légende certains ont dû la faire pour être acceptés, d’autres non lol ) Lors de ses quêtes, diverses notions du développement sont données au wilders, comme le concept du “wireframe”, CodeSandBox, les bases du HTML et CSS et même une introduction à JavaScript.
